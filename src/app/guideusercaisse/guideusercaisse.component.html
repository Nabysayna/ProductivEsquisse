Introduction #0

Bienvenue sur le guide d’utilisation de votre plateforme Sentool. Ce support vous montre en détail l’utilisation de votre outil du futur.

Vous trouverez toutes les informations nécessaires pour mieux comprendre votre outil et l’utiliser dans toutes ses potentialités.

Postecash #1

Allez sur l’onglet PosteCash pour accéder à l’interface réservée aux services de PosteCash

  1. / Achat de code woyofal

    Cliquez sur l’onglet « Achat de code woyofal » puis compléter les informations demandes et cliquer sur Valider

  2. / Achat de carte JULA

    Cliquez sur l’onglet « Achat de code woyofal » puis compléter les informations demandes et cliquer sur Valider

  3. / Recharger une carte PosteCash espèces

    Remplir les champs vides avec les informations demandés puis cliquer sur Valider

  4. / Règlement de facture Senelec en espèces

    Remplir les champs vides avec les informations requises puis cliquer sur Valider

TNT by Excaf #4

Allez sur l’onglet TNT by Excaf pour accéder à l’interface réservée aux services de TNT

  1. / Vente un nouvel abonnement

    Sélectionner le service TNT et entrer les informations du client puis cliquer sur Valider

  2. / Vente de carte

    Remplir les informations sur les champs requis et cliquer sur Valider

  3. / Vente de décodeurs

    Remplir les champs requis avec les informations du client et appuyer sur Valider

  4. / Suivis Abonnement, décodeur et cartes

    Liste des abonnements, de vente décodeur et de vente de decodeurs
